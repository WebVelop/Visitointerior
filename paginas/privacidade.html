INFORMAÇÃO LEGAL

A proteção dos seus dados pessoais é importante para o VisitoInterior. Esta Política de Privacidade explica como recolhemos, utilizamos e protegemos as informações dos utilizadores do nosso website.

1. Responsável pelo tratamento dos dados

O website VisitoInterior é responsável pelo tratamento dos dados pessoais recolhidos através deste website, em conformidade com o Regulamento Geral sobre a Proteção de Dados (RGPD).

2. Dados recolhidos

Podemos recolher informações fornecidas voluntariamente pelos utilizadores, nomeadamente através de formulários de contacto, subscrição de newsletters ou outros meios disponibilizados no website.

Poderão ainda ser recolhidos dados técnicos, como endereço IP, tipo de navegador, dispositivo utilizado e informações estatísticas de navegação.

3. Finalidade da recolha

Os dados recolhidos destinam-se exclusivamente a:

  • Responder a pedidos de contacto;
  • Melhorar a experiência de navegação;
  • Enviar informações solicitadas pelo utilizador;
  • Cumprir obrigações legais aplicáveis.

4. Conservação dos dados

Os dados pessoais serão conservados apenas durante o período necessário para cumprir as finalidades para as quais foram recolhidos ou para cumprimento de obrigações legais.

5. Partilha de dados

O VisitoInterior não vende nem cede dados pessoais a terceiros, exceto quando tal seja exigido por lei ou necessário para a prestação de serviços associados ao funcionamento do website.

6. Cookies

Este website poderá utilizar cookies para melhorar o desempenho e a experiência de navegação. O utilizador poderá configurar o seu navegador para recusar ou eliminar cookies a qualquer momento.

7. Direitos do utilizador

Nos termos da legislação aplicável, o utilizador tem o direito de aceder, retificar, atualizar, limitar ou eliminar os seus dados pessoais, bem como retirar o consentimento previamente concedido.

8. Segurança

São adotadas medidas técnicas e organizativas adequadas para proteger os dados pessoais contra perda, utilização indevida, acesso não autorizado ou divulgação.

9. Alterações a esta política

O VisitoInterior poderá atualizar esta Política de Privacidade sempre que necessário. Recomenda-se a consulta periódica desta página.

10. Contacto

Para qualquer questão relacionada com esta Política de Privacidade ou com o tratamento de dados pessoais, poderá contactar-nos através dos meios disponibilizados na página de contacto.
